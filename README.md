## What's this? ##

This is a repo of http://trefmanic.me, which is my blog. Tools used to create this blog:

- [Jekyll][1]
- [jekyll-now][2] by [Barry Clark][3]
- [jScrollPane][4] for custom scrollbars (it's [here on GitHub][5] too)

[2]: https://github.com/barryclark/jekyll-now "jekyll-now template"
[3]: https://github.com/barryclark "Barry Clark's GitHub profile"
[4]: http://jscrollpane.kelvinluck.com/ "jScrollPane website"
[5]: https://github.com/vitch/jScrollPane "jScrollPane GitHub repo"
